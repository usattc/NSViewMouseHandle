# NSViewMouseHandle
A NSView category for determine mouse entered or exited in masOS Apps.
<br />
NSView and subclass of NSView can use it.
<br />
It's easy to use.
<br />
Sample Code:
[button setMouseExited:^{<br />
       // Your code...<br />
 }];
 <br />
 [button setMouseEntered:^{<br />
       // Your code...<br />
 }];
