# NSViewMouseHandle
A NSView category for determine mouse entered or exited in masOS Apps.
<br />
<br />
NSView and subclass of NSView can use it.
<br />
<br />
It's easy to use.
<br />
<br />
Sample Code:

```Obeject-C
[button setMouseExited:^
       // Your code...
 }];

 [button setMouseEntered:^{
       // Your code...
 }];
 ```
